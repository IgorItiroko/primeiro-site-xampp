### Site com lista de Filmes ###
Por: Igor Shinji Itiroko

#------------------------------#
    Seguem na versão os arquivos .php, .css e .sql para rodar
o site em um servidor local.
Sua interface inicial mostra os filmes que estão registrados 
no banco de dados, ao clicar neles é aberto uma página mais detalhada
sobre cada um, com sinopse, avaliação e gênero do filme.

    Para alterar essa lista é necessário logar como administrador, indicado
na interface como "Logar Admin", o qual o redireciona para um autenticador
que solicita o usuario e senha de acesso, também controlado por banco
de dados. Inicialmente o usuario é "igorshinji" e a senha "asdf".

    Na interface de administrador há uma visualização mais simples dos 
filmes da lista feito em forma de tabela, nessa interface é possível 
adicionar novos filmes, editar o conteúdo dos filmes já existentes e
excluir um filme.
